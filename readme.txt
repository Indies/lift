1) goto http://maven.apache.org/ an install Maven 2
2) goto the root directory of Scala with Sails (that's were the readme.txt file is located)
3) type "mvn install" in your command line (without the quotes)
3) goto the "scalawithsails-example" and type "mvn jetty:run"
4) open a webrowser "http://localhost:8888/"

That's it :)
